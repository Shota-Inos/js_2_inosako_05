Twitter風に作成 下記順序にて、操作いただけますでしょうか 
１、なまえテキストエリアにて、名前を入力 
２、決定ボタンをクリック（sessionstorageにて保存） 
３、Let's Delployをクリック 
４、左側リスト内から、Deployをクリックする 
５、テキストエリアに文字を入力する
６、任意のSaveボタンをクリックする（Save Diaryをクリックすると、日記タブにデータが蓄積される）
